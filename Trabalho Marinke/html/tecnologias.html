Tabela de Tecnologias e Linguagens

Tecnologia/Linguagem Descrição
HTML Linguagem de marcação utilizada para estruturar o conteúdo da web.
HTML5 Versão mais atualizada do HTML, com novos elementos semânticos e funcionalidades aprimoradas.
CSS Linguagem de estilo utilizada para definir a aparência visual das páginas web.
CSS3 Atualização do CSS que traz recursos modernos como animações, transições e gradientes.
