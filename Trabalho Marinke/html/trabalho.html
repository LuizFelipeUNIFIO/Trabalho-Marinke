Trabalho Marinke

O que é: Servidor FTP, JavaScript, Docker.

Servidor FTP

Software que utiliza o protocolo FTP para transferir arquivos entre computadores via internet ou redes locais. Permite uploads, downloads, autenticação de usuários e registro de atividades. Seu desenvolvimento tem raízes antigas, com a especificação inicial sendo atribuída a Abhay Bhushan (RFC 959).

JavaScript

Linguagem de programação essencial para a web, criada por Brendan Eich em 1995. Permite a criação de páginas interativas e dinâmicas, manipulando HTML e CSS em tempo real. Seu padrão, ECMAScript, impulsionou o surgimento de bibliotecas e frameworks modernos, ampliando o uso tanto no front-end quanto no back-end.

Docker

Plataforma de virtualização baseada em contêineres, desenvolvida por Solomon Hykes e lançada em 2013. Permite empacotar aplicações com todas as suas dependências, garantindo portabilidade e consistência em qualquer ambiente. Ferramentas como Docker Compose e Docker Swarm auxiliam na gestão e orquestração dos contêineres.
